# Proyecto-cyber
Vamos a desarrollar una herramienta por consola que permite descifrar contraseñas a partir de su hash usando diferentes técnicas de criptoanálisis. El sistema ofrecerá al usuario la posibilidad de seleccionar entre ataques de fuerza bruta y diccionario, así como un modo automático que decidirá el método más adecuado en función de las características de la contraseña. 

La herramienta mostrará métricas como el tiempo de ejecución y el número de intentos realizados, permitiendo comparar la eficiencia de los distintos enfoques. El objetivo es aplicar de forma práctica los conceptos de criptografía y ataques vistos en clase, analizando el rendimiento y las limitaciones de cada técnica. 

Asier: que el usuario pueda ejecutar el codigo en terminal y el programa pregunte donde quiere lanzar el codigo (un hash que copiamos, en instagram/su pagina web, …) y despues escoger a fuerza bruta o por diccionario. si es fuerza bruta: solo caracteres, caracteres con numeros, etc. combinaciones posibles basicamente. y luego que enseñe el tiempo que va a necesitar (numero de intentos calcular).
